项目前端是基于 Ant Design Pro 来创建的，后端是基于 go-zero 来创建的一个前后端分离的管理系统

[如果喜欢 Element Admin UI 前端版本的，可以点击查看](https://github.com/feihua/go-zero-admin-vue)

# 1.安装 node_modules:

```shell
npm install
```

# 2.启动项目

```shell
npm run dev
```
